# relational db

using cpp-ethereum merkle patricia tree

## compile

`chmod +x build_server_client.sh`

`./build_server_client`

## compilatio for linux

change `lboost_thread-mt` to `lboost_thread`

change `-std=c++1z` to `-std=c++17`

## run

`./server_variadic`

on multiple terminals

`./client_variadic`
